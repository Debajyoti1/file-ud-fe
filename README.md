# File UD Frontend

This project is a **React application built with Vite**.

---

## Run the Project

1. **Configure Environment Variables**  
   - Rename the `env` file to `.env`  
   - Open `.env` and set the correct backend URL, for example:

   ```env
   VITE_API_URL=http://your-backend-url

2. **Install Dependencies**

    - npm install

3. **Run the project**

 - npm run dev
    The app will start in development mode. Open your browser at http://localhost:5173 to view it.
